## Quick Start (Local)

### Prerequisites
- Python 3.10+
- pip

### Step 1 — Install dependencies
```bash
cd backend
pip install -r requirements.txt
```

### Step 2 — Train the model
```bash
python train_model.py
# Outputs: backend/model_artifact.pkl
# Expected: MAE ≈ 8–15 MW, MAPE ≈ 2–4%
```

### Step 3 — Start the API
```bash
python main.py
# Or run: uvicorn main:app --reload --port 8000
```
Running `python main.py` prints a clickable link directly in the terminal.
API docs: http://localhost:8000/docs

### Step 4 — Open the dashboard
Open your browser and visit:
 **[http://localhost:8000/](http://localhost:8000/)**

*Note: The backend now serves the frontend dashboard natively on the root route, so you do not need to open index.html as a local file or run a separate HTTP server.*

## Data & Modeling Decisions

### Data Cleaning
- **Negative values** → set to NaN (physically impossible load readings)
- **Outliers** → IQR method (1st–99th percentile ± 3×IQR), replaced with NaN
- **Gaps** → linear interpolation, max 6 consecutive (= 1 hour), then drop remaining NaN

### Weather Source
**Open-Meteo** (`https://api.open-meteo.com`) — free, no API key, returns temperature, humidity, cloud cover, wind speed for Dhanbad (23.7957°N, 86.4304°E).

### Localized Holidays (Dhanbad, Jharkhand)
Beyond generic national holidays, we include:
- **Jharkhand state holidays**: Foundation Day (Nov 15), Karma Puja, Sarhul (tribal)
- **Industrial shutdowns**: BCCL and ECL coal mine annual maintenance days
- **Local festivals**: Tusu Puja (Makar Sankranti form specific to Jharkhand)

### Model: GradientBoostingRegressor
Chosen over LSTM and ARIMA because:
1. Our EDA shows clear feature-based patterns (not long-range temporal dependencies)
2. Natively handles all tabular features (weather, calendar, holidays)
3. No normalization required; built-in feature importance for explainability
4. Consistently outperforms LSTM on structured tabular time-series (M5 competition benchmark)

**Features (18 total):** hour, minute, day_of_week, month, day_of_year, is_weekend, block_of_day, temperature, humidity, cloud_cover, wind_speed, is_holiday, + cyclical sin/cos encodings for hour, day_of_week, month.
